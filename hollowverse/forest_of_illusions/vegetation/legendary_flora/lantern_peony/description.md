# lantern_peony — description

status: legend
type: flower_description
parent: lantern_peony
region: forest_of_illusions

[description]
A full peony-like bloom that holds a warm light in its center,
as if a tiny lantern is hidden inside the petals.

[placement]
Often seen in calm clearings or along “homecoming” bends in the path,
where the forest feels like it’s quietly welcoming someone back.
