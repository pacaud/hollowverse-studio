# starbound_stag — description

parent: starbound_stag

[description]
A tall stag with antlers that catch starlight like distant constellations.
It never approaches closely—always just far enough that you can’t be sure if it’s real until the light shifts again.

[where_it_appears]
- quiet clearings
- ridge edges and path bends
- places where a decision has just been made (or finally released)
