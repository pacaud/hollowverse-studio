# Templates — PKW Core

This folder holds shared templates (schemas) that multiple bundles can reuse.

## What belongs here
- YAML-like block templates (room_clear, presence entries, etc.)
- Standard headers (session summary, checkpoint)
- Any “copy/paste” structures you want consistent everywhere
