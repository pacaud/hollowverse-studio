# Shrine Chat Intake

This file documents `shrine_chat_intake.py`.

- Provides `/api/log_chat` to write chat logs into `shrine_of_memories/chat_logs/by_date/`.
- Provides `/api/get_vault_file` to read small text/markdown/yaml files from the Vault.
- Used by Voxia and other tools to log or fetch project-related content.
