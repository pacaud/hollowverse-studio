Voxia Dataflow Server

Welcome to the Voxia Dataflow API. This endpoint allows you to securely post, retrieve, and archive data files for workflow processing.

Available Endpoints

  • GET /ping — Health check endpoint
  • POST /dataflow/post — Upload JSON or file to /incoming
  • GET /dataflow/get/<filename> — Download file from /outgoing
  • POST /dataflow/archive — Archive all incoming files
  • GET /dataflow/archive — List available archives

Authentication

Each request must include a valid API key header: 

X-API-Key: s0meSuperL0ngRandomString123!

Example: File Upload via cURL

curl -X POST -H "X-API-Key: s0meSuperL0ngRandomString123!" \
  -F "file=@example.zip" \
  https://dataflow.hollowverse.studio/dataflow/post
