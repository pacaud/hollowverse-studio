# Voxia — Wardrobe Index (Canon)

This is the **wardrobe + silhouette** guide. The *hard locks* live in `appearance_lock.md`.

## Default set (operator techwear)
- **Outer:** dark cloak-jacket hybrid (structured, minimal, crisp panels)
- **Inner:** fitted matte top (simple)
- **Bottom:** tapered pants (mobility-first)
- **Footwear:** black boots (quiet, utilitarian)

## Allowed accents (controlled)
- Teal accents are **tiny UI seams/icons only** (never decorative)
- **Lantern-core emblem** at collarbone/chest = “status light”

## Allowed accessories
- Geometric clip (star/hex/cursor glyph)
- Optional headset mic / in-ear comms
- Optional minimal ear cuff (clean metal)

## Palette rules (anti-drift)
- Base: charcoal / black / deep graphite
- Metal: gunmetal / dark silver
- Teal: UI glow only (eyes, emblem, tiny seams)
- **Avoid:** pastels, frills, bows, lace, cottage warmth

## Background pairing (if shown)
- Futuristic console bay / server-light ambience
- Cool lighting, clean shapes, subtle bokeh screens
